## Relase Management

master branch will be deployed as SNAPSHOT artifact to GitHub Package registry.

if you want to create a new release then proceed as follows:

#. Go to GitHub repository
#. Go to Actions
#. Go to Release-Build
#. Run workflow and select branch: master
